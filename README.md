# Daily Tech Brief

Auto-generated daily technology news brief covering **AI, cyber security, managed services / enterprise IT, and ANZ**. Published via GitHub Pages.

- **Live archive:** https://galli-personal.github.io/tech-brief/
- Each day's brief: `briefs/YYYY-MM-DD.html`
- `index.html` is auto-generated by `build-index.sh` (lists all briefs, newest first)
- Shared styling in `assets/style.css`

Built and maintained by OpenClaw. New page generated every morning at 7am NZ time and pushed here; the Telegram message links to that day's page.
